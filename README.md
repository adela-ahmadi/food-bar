# FoodBar

A modern, responsive Fast-Food & Food Delivery website template built with Bootstrap 5 and custom CSS. This project features high-performance UI components tailored for restaurants, burger shops, and food delivery services.

## Live Demo

- [https://adela-ahmadi.github.io/food-bar/]

## Core Features

- **Multi-Page Layout**: Includes index, shop, product details, cart, checkout, contact, and custom 404 pages.
- **Dynamic Menu Filtering**: Responsive tabbed navigation for quick switching between food categories (Burgers, Pizzas, Fried Chicken, etc.).
- **Functional Modals**: Full-screen interactive search overlay independent of the main navigation flow.
- **Smooth Animation Effects**: Tailored transition curves applied to the testimonial sliders for enhanced UI/UX.
- **Optimized E-commerce UI**: Structured billing forms, order summary tables, and custom circular pagination elements.

## Technologies Used

- **HTML5** & **CSS3**
- **Bootstrap v5.3** (Responsive Grid & Components)
- **FontAwesome v6.5** (Vector Icons)
- **Google Fonts** (Poppins Typography)

## Getting Started

### Prerequisites

You only need a modern web browser to run this project locally.

### Installation

1. Clone the repository:
   ```bash
   git clone https://github.com
   ```
2. Navigate to the project directory:
   ```bash
   cd food-bar
   ```
3. Open `index.html` in your favorite browser.

## Customization

To update the core theme color, modify the hex values (default orange: `#ff8f00`, dark grey: `#344042`) inside the `style.css` file.
